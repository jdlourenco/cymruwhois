DESCRIPTION:

cymruwhois is a simple Ruby module that utilizes Team Cymru’s IP to ASN Mapping via DNS queries. The module was conceived as an IP geolocation alternative to GeoIP.

Kindly take note of Team Cymru’s "Special Notice" for bulk queries at the top of the page: http://www.team-cymru.org/Services/ip-to-asn.html

FEATURES/PROBLEMS:

The module does not include the Bogon, and Malware Hash registry services of Team Cymru.

SYNOPSIS:

require ‘cymruwhois’

c = Cymru::IPAddress.new c.whois("72.4.120.124") > ["27357", "72.4.112.0/20", "US", "ARIN", "2008-10-09", "RACKSPACE - RACKSPACE HOSTING"] c.country > "US"

as = Cymru::ASNumber.new as.whois("27357") > ["27357", "US", "ARIN", "2003-02-20", "RACKSPACE - RACKSPACE HOSTING"]
